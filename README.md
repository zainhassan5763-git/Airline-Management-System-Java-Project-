# Airline Management System

A Java-based desktop application built as part of my Object-Oriented Programming course at SSUET (Sir Syed University of Engineering & Technology).

## About

This project simulates a real-world airline management system where an admin can manage flights and passengers, and passengers can book or cancel tickets — all through a clean GUI built with Java Swing.

## Features

### Admin Panel
- Secure admin login
- Add and search flights
- Add passengers with photo
- View passenger details and photo

### Passenger Portal
- Passenger login with ID
- Book flight tickets
- Search tickets (displayed as boarding pass)
- Cancel tickets

## Technologies Used

- **Java** — Core programming language
- **Java Swing** — GUI framework
- **OOP Concepts** — Inheritance, Encapsulation, Abstraction, Polymorphism

## How to Run

1. Clone this repository
2. Open in **IntelliJ IDEA**
3. Mark `src/` as Sources Root
4. Place the `assets/` folder in the project root
5. Run `AdminGUI.java`
